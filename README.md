# User Profile Card Generator

## Description

A Flask-based web application that allows users to generate a profile card by entering their name, bio, and profile image URL.

## Features

* Enter user details through a form
* Generate profile cards dynamically
* Display profile image, name, and bio
* Responsive and modern user interface
* Flask backend handling POST requests

## Technologies Used

* Python
* Flask
* HTML
* CSS

## How to Run

1. Install Flask

pip install flask

2. Run the application

python app.py

3. Open in browser

http://127.0.0.1:5000

## Project Structure

* app.py
* templates/index.html
* static/style.css
